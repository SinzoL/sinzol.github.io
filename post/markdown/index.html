# Markdown 完整指南：从高频使用到低频使用

Markdown 是一种轻量级标记语言，它允许你用纯文本格式化文档，方便地进行网页内容创建、博客写作、文档生成等。今天，我们将按使用频次从高到低，介绍 Markdown 的常用语法和示例，帮助你更高效地使用它。

## 1. 标题

### 用法：
Markdown 提供了六级标题，使用 `#` 符号来定义标题的等级，`#` 的数量决定标题的层级。

### 示例：
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

输出：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2. 强调（加粗和斜体）

用法：

  • 加粗：使用两个星号 ** 或两个下划线 __ 包围文本。
  • 斜体：使用一个星号 * 或一个下划线 _ 包围文本。

示例：

**这是加粗文本**
__这也是加粗文本__

*这是斜体文本*
_这也是斜体文本_

输出：

这是加粗文本
这也是加粗文本

这是斜体文本
这也是斜体文本

3. 列表

用法：

Markdown 支持有序列表和无序列表：

  • 无序列表：使用 -、+ 或 *。
  • 有序列表：使用数字加点（如 1.、2. 等）。

示例：

- 苹果
- 香蕉
- 橙子

1. 第一步
2. 第二步
3. 第三步

输出：

  • 苹果
  • 香蕉
  • 橙子
  1. 第一步
  2. 第二步
  3. 第三步

4. 链接

用法：

Markdown 允许你插入链接，有两种方式：

  • 行内链接：[链接文本](链接地址)
  • 引用式链接：[链接文本][id] 和在文档的其他地方定义该链接 id。

示例：

[Google](https://www.google.com)

[GitHub]: https://github.com
访问 [GitHub](https://github.com) 获取更多信息。

输出：

Google
[GitHub]: https://github.com
访问 GitHub 获取更多信息。

5. 图片

用法：

插入图片的语法与链接类似，只不过需要在前面加一个 ! 符号。

示例：

![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)

6. 代码

用法：

  • 行内代码：使用反引号 ` 包围代码。
  • 代码块：使用三个反引号 ``` 来包围多行代码。

示例：

这是 `行内代码` 示例。

```python
def hello_world():
    print("Hello, World!")


### 输出：
这是 `行内代码` 示例。

```python
def hello_world():
    print("Hello, World!")

7. 引用

用法：

引用文本使用 > 符号。

示例：

> 这是一个引用示例。
> 
> Markdown 允许你轻松创建引用块。

输出：

这是一个引用示例。

Markdown 允许你轻松创建引用块。

8. 水平线

用法：

使用三个或更多的 -、* 或 _ 创建水平线。

示例：

---
或
***
或
___

输出：

或

或

9. 表格

用法：

使用管道符 | 来创建表格，并用破折号 - 来分隔表头和表格内容。

示例：

| 姓名  | 年龄 | 性别 |
| ----- | ---- | ---- |
| 小明  | 25   | 男   |
| 小红  | 22   | 女   |
| 小亮  | 24   | 男   |

输出：

姓名 年龄 性别
小明 25 男
小红 22 女
小亮 24 男

10. 任务列表

用法：

Markdown 支持任务列表，使用 - [ ] 来表示未完成任务，- [x] 来表示已完成任务。

示例：

- [x] 完成博客写作
- [ ] 进行代码审查
- [ ] 部署网站

输出：

11. 嵌入 HTML

用法：

Markdown 支持直接嵌入 HTML 标签。如果需要更复杂的布局或元素，可以使用 HTML。

示例：

<p style="color:blue;">这是一个蓝色的段落</p>

输出：

这是一个蓝色的段落

12. 注脚

用法：

通过使用 [^1] 语法来创建注脚。

示例：

这是一个带注脚的文本[^1]。

[^1]: 这是注脚内容。

输出：

这是一个带注脚的文本[1]。

结论

Markdown 是一个非常强大的标记语言，适用于从简单的文本到复杂的网页内容的各种场景。通过本文的介绍，你可以掌握常用的 Markdown 语法，并将其应用于你的博客、文档或任何其他的在线内容创建中。希望你能快速掌握并开始使用 Markdown 来提升你的写作和工作效率！
