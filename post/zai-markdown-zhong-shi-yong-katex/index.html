2024-12-14

在 Markdown 中使用 KaTeX

如果你的 Markdown 编辑器或博客平台支持 KaTeX，你可以直接在 Markdown 中书写 LaTeX 语法，KaTeX 会自动渲染公式。

行内公式

要在行内插入公式，可以使用单个美元符号 $ 将公式包围起来。例如：

这是一个行内公式：$E = mc^2$

输出：
这是一个行内公式：( E = mc^2 )

块级公式

要插入块级公式，可以使用双美元符号 $$ 将公式包围起来。例如：

$$
E = mc^2
$$

输出：

[
E = mc^2
]

4. 常见的 LaTeX 数学语法

KaTeX 支持绝大多数的 LaTeX 数学命令，以下是一些常见的数学公式示例。

分式公式

这是一个分式公式：$\frac{a}{b}$

输出：
[
\frac{a}{b}
]

根号公式

这是一个根号公式：$\sqrt{a^2 + b^2}$

输出：
[
\sqrt{a^2 + b^2}
]

求和公式

这是一个求和公式：$\sum_{i=1}^{n} a_i$

输出：
[
\sum_{i=1}^{n} a_i
]

极限公式

这是一个极限公式：$\lim_{x \to \infty} f(x)$

输出：
[
\lim_{x \to \infty} f(x)
]
